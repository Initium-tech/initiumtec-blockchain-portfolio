Everything **you personally** must do so these assets can be sold and so other
people have real reasons to give them value. The code is done; from here on,
value is created by *your* actions, not more programming.

people want to buy it, and people only buy for two honest reasons — the
mission/meme speaks to them, or the token does something for them. Every task
below builds one of those two reasons.

## Phase 5 — After launch: converting value to dollars (the sell mechanics)

How money legitimately reaches you — and every step is public by design:

1. **LP fees** accrue on every trade (0.3%) → claim → swap to ETH
2. **Team allocation sales**: announced ≥72h ahead, within the published 10%,
   never more
3. **ETH → Coinbase → USD**: send ETH to your Coinbase deposit address
   (small test send first), sell, withdraw. Log every transaction for taxes.
4. **Quarterly conservation donations**: send, publish the tx hash, post it.
   This is also what keeps the story alive and the buyers proud.

## Rules that protect everything (read monthly)

1. Never promise profit — not in posts, DMs, or "jokes." (Legal + ethical.)
2. Never sell unannounced. The published schedule is the contract with your
   community.
3. One canonical contract address, announced once, from one account.
4. Every donation on-chain and published. A missed one ends the project's
   credibility.
5. Private keys: hardware wallet for real funds, never typed into any
   website, never photographed, never shared — including with Claude.
6. If it stops being fun/true, wind it down publicly. An honest ending
   preserves Initiumtec's name; a quiet abandonment doesn't.
